# Boilerplate: Express with Typescript

## Install

```sh
npm i
```

## Development

```
npm run dev
```
Open the browser and navigate to http://localhost:3000

## Build
To compile the code from Typescript to Javascript run:
```
npm run build
```

The compiled code will be generated under `/dist`
